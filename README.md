# منصة توثيق شهادات نقابة تكنولوجيا المعلومات

## 🏛️ لجنة التحول الرقمي والتدريب

### 📋 عن المنصة
منصة متكاملة لتوثيق وإدارة شهادات الدورات التدريبية الصادرة عن الأكاديميات المعتمدة من نقابة تكنولوجيا المعلومات والبرمجيات.

### ✨ المميزات الرئيسية

#### 🔍 البحث والتحقق
- البحث عن الشهادات باستخدام الكود الفريد
- البحث عن الأكاديميات باستخدام الرقم الاعتمادي
- عرض بيانات الأكاديمية وفروعها في 27 محافظة
- تسجيل بيانات الباحث (الاسم، رقم التليفون، الجهة، المحافظة)
- طباعة وتحميل الشهادة بصيغة PDF مع لوجو وختم النقابة

#### 👨‍💼 لوحة التحكم
- **سوبر أدمن**: صلاحيات كاملة (إضافة، تعديل، حذف، إدارة المستخدمين)
- **أدمن**: صلاحيات (إضافة، تعديل) فقط - لا يمكنه الحذف
- تسجيل كل نشاط (من قام به، متى، من أي جهاز)
- سجل متكامل للبحث (مين بحث، امتى، على ايه)

#### 📊 التقارير
- تقارير تفصيلية عن الشهادات والأكاديميات
- تصدير PDF أو Excel مع لوجو النقابة
- إحصائيات فورية في لوحة التحكم

### 🛡️ الأمان
- تشفير البيانات الحساسة
- حماية المسارات (Route Protection)
- منع التنقل باللينك بدون صلاحية
- تسجيل الدخول الآمن باستخدام Firebase Auth
- حماية ضد هجمات XSS, CSRF, SQL Injection

### 🚀 التقنيات المستخدمة
- **Next.js 14** - الإطار الرئيسي
- **TypeScript** - الأمان والتنظيم
- **Firebase** - قاعدة البيانات والمصادقة
- **Cloudinary** - رفع الصور والملفات
- **Tailwind CSS** - التصميم
- **React PDF** - توليد الشهادات
- **XLSX** - تصدير Excel
